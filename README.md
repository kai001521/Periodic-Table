# Interactive Periodic Table

A modern and interactive Periodic Table of Elements built using HTML, CSS, and JavaScript.

The application provides an engaging way to explore all 118 chemical elements through an intuitive interface, category filters, and a unique 3D globe visualization.

---

## Features

- Complete Periodic Table (118 Elements)
- Interactive Element Information
- 3D Globe View
- Category Filters
- Responsive Design
- Mobile Landscape Optimization
- Glassmorphism User Interface
- Color-coded Element Categories
- Fast and Lightweight
- Built with Vanilla JavaScript

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/interactive-periodic-table.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

No installation or dependencies required.
